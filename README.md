# simple-asana
An Asana app clone with Vanilla Javascript ES6

### Running the project
Open the project
```sh
cd simple-asana
```

Install a http server
```sh
npm install http-server -g
```

Start http server
```sh
http-server
```

Install SASS
```sh
sudo gem install sass
```

Run SASS task
```sh
sass --watch sass:dist
```

Access `http://localhost:8080`
